## 🎮 How to Play
- Use arrow keys to control the snake.
- Eat the food to grow longer.
- Don't hit the walls or yourself!

## 🚀 Features
- Classic snake game mechanics
- Simple and responsive design
- Score tracking

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
